Pipes Decimales y Porcentaje

Estan incluidos en el Common Module de Angular

  • ventasNetas | number
  • ventasNetas | number : "1.2-2"
  • ventasNetas | number : "1.0-0"
  • {{ventasNetas | number }}
  • {{ventasNetas | number : "1.2-2" }}
  • {{ventasNetas | number : "1.0-0" }}
  • ventasNetas | currency
  • ventasNetas | currency: "Peso"
  • ventasNetas | currency: "US":"symbol-narrow"
  • {{ventasNetas + 2000000 | currency }}
  • {{ventasNetas + 2000000 | currency: "Peso" }}
  • {{ventasNetas + 2000000 | currency: "US":"symbol-narrow" }}
  • porcentaje | percent
  • porcentaje | percent:"2.2-2"
  • {{porcentaje | percent }}
  • {{porcentaje + .0056 | percent:"2.2-2" }}
